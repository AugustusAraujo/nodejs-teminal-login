# NODE TERMINAL LOGIN

## Este é um login via terminal usando Typescript e a biblioteca InquirerJs

## Adicionar usuários

### Adicione um objeto no arquivo `users.json`

### Exemplo:

```json
{
    "username":<USERNAME>,
    "password":<PASSWORD>
}
```
## Usando

### Primeiro instale as dependências com `npm install`

### Depois user `npx ts-node index.ts`
